RX580 2048sp刷570 VBios

2020-05-22 Views 黑苹果 经验

显卡已出，年度理财产品！

已入2060 美好夜之城~

显卡特性

  • 盈通RX580 2048sp大地之神8GB
  • 输出接口 dvi*2个

    *貌似新版显卡是dvi*1 / DP*3

准备软件

  • Atiflash

刷入Vbios

  1. cd Atiflash文件夹路径；
  2. 依次命令；
amdvbflash.exe -unlockrom 0 #解锁显卡BIOS
ROM Unlocked #说明 Bios 解锁成功
amdvbflash -f -p 0 XXXX.rom #XXXX替换为要刷入的ROM名
Restart System To Complete VBIOS Update #即为完成；重启即可
  • 注释：XXXX为你下载的VBIOS文件名，也可以将下载好的文件名改成简单的名称即可。譬如 0.ROM
  • 下载好的ROM需要放在刷BIOS文件夹目录下！
  • 重启后如遇设备管理器不显示显卡，更新驱动即可。
